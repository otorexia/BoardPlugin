# Board Setup

# Types of arguments

You can pass two type of arguments in board (beside scene which is required as first argument). They are 'Global' and 'Items'.

# Global

Global is used for control of all board under that instance. To pass a global argument object, there has to be a type parameter set to global.

Global arguments is optional, however it's needed to control behavior of board, both on creating and controlling of it's items or the board in general.

example of a global settings object will look like this.

{
  type: 'global',
  isSelectable: true,
  selectAgainst: 1,
  matchAgainst: 1,
  getMatch: 'all'
}

# Items

Items argumetns are for generating boards and it's required to atleast pass an empty object to create a board. You can pass this type of arguments as many as you like, It'll simply make boards equal to number of arguments you pass. there's no need of type parameters in items arguments.

example of a items argument.

{
  cols: 6,
  rows: 6,
  items: 'items',
  skipValue: true,
  size: 50,
  maxSize: {
    width: 50,
    height: 100
  }
}
